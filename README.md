# 网红模拟经营

一款基于 HTML/CSS/JavaScript 的网红模拟经营游戏。

## 游戏特色

- 📱 多平台适配：PC 和移动端均可流畅游戏
- 🎭 四大类别：科普、吃播、美妆、生活，各具特色
- 📊 属性系统：颜值、学历、幽默、气质、镜头感
- 💰 经营要素：存款管理、团队建设、内容矩阵
- 🎯 职级晋升：从素人到 MCN 签约，体验成长
- 📅 真实事件：结合 2026-2027 年时间线的行业事件

## 如何开始

直接打开 `index.html` 文件即可开始游戏。

## 游戏说明

详见游戏内"说明"按钮。

## 技术栈

- 纯前端实现
- LocalStorage 存档系统
- 响应式 UI 设计
