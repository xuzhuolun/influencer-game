========================================
网红模拟经营 - 怎么分享给其他人玩
========================================

【推荐】方式一：发一个链接（别人点开就能玩）
----------------------------------------
把游戏放到网上，然后发链接给朋友即可。

1. 用 GitHub 发布成网页（免费、稳定）：
   - 按《使用说明-GitHub分享.txt》把项目推到 GitHub 并开启 Pages
   - 开启后你会得到一个地址，例如：
     https://你的用户名.github.io/influencer-game/
   - 把这个链接发给别人，他们用手机或电脑浏览器打开就能玩

2. 若已推送过代码，只需在 GitHub 仓库里：
   - 点 Settings → Pages
   - Source 选 "Deploy from a branch"
   - Branch 选 main，文件夹选 / (root)
   - 保存后等 1～2 分钟，同一地址即可访问


方式二：同一 WiFi 下，用局域网地址一起玩
----------------------------------------
适合在办公室/家里和身边人一起玩。

1. 双击运行：start-server.bat
   （需要电脑已安装 Python，没有的话去 python.org 下载）

2. 本机浏览器打开：http://localhost:8080

3. 查本机局域网 IP（在命令行输入 ipconfig，看 IPv4 地址）
   把链接发给同一 WiFi 下的朋友，例如：
   http://192.168.1.100:8080
   他们用手机或电脑浏览器打开这个地址即可

注意：关掉你的电脑或关掉脚本，别人就访问不了了。


方式三：直接发整个游戏文件夹
----------------------------------------
1. 把整个 D:\wanghong 文件夹打包成 zip 发给对方
2. 对方解压后，双击里面的 index.html 用浏览器打开

注意：部分浏览器对“本地文件”限制较严，可能出现打不开或功能异常，
      更推荐用方式一（发链接）或方式二（局域网地址）。


总结：最省事的是方式一，发一个 GitHub Pages 链接，谁都能点开玩。
